git add ./           ---------------添加文件
git commit -m "描述"     --------------将添加的文件提交到仓库
git status           --------------查看git状态，是否有修改的文件未添加，添加的文件未提交
git reset            -------------重置git状态，可以让git返回以前提交的版本
git reflog           ------------查看提交过的版本
git branch          -------------git 创建分支，
git checkout -b         -------------创建分支并切换
git checkout        -------------切换到分支
git merge           ----------------合并分支到最新版本，合并有冲突时，需要手动处理，再次添加提交
